This program generates the MandelBrot set.

for each pixel it calculates it value of brightness
and puts it into an array.

It does 500 iteration to check for every pixel.
Each threads handles one pixel and threads are arranged in a 16x16 grid
The image it generates is of N x N where N = 4192.

Finally python is used to visualize it.
